Tutorial: Python

Prerequisites

  • Install Ibis Birdbrain

Overview

You can use Ibis Birdbrain as a Python library, but it is often used through an interactive Python session. You can see how to start one with the birdbrain CLI with Ibis Birdbrain ready to use.

If you use the CLI, you’ll have access to the ai/bot/birdbrain object, which is an instance of the ibis_birdbrain.ai.Bot class. This bot has access to numerous tools and can be used to interact with data via Ibis.

Getting help

You can ask the bot for help with its usage:

bot("how do I use you?")

birdbrain:

I’m birdbrain, your assistant for data analysis projects using Ibis, AI, and many other data tools. Here’s how you can interact with me:

  1. Ask Questions: You can ask me anything related to Ibis, AI, and data analysis. For example, you can ask about how to use certain features in Ibis, how to code in Python, or ask for help in debugging your code.

  2. Run Analysis: You can also ask me to run data analysis for you. You can give me instructions to query data, analyze it, and even generate visualizations.

  3. Learning Resources: If you’re looking for resources to learn more about Ibis, AI, and data analysis, I can provide you with relevant links to documentation, tutorials, and more.

  4. File Operations: I can help with file operations such as reading a file, writing to a file, listing files in a directory, etc.

  5. Translation & Summarization: If you have a large piece of text or a document that you want to summarize, or if you want to translate text from one language to another, I can help with that too.

Remember, I can only provide accurate information and perform tasks based on the data I have access to. I’m here to assist you. Let’s explore the world of data together!

Interacting with a data

Ibis Birdbrain can read data files (limited to Delta Lake tables and Excel files, currently), registering them as tables in your Ibis backend. If you connect to a data platform with existing tables via Ibis, you will have access to them through the bot.

In this example, we’ll read in a local penguins.xlsx file and ask the bot about it:

bot("read in the penguins excel")

birdbrain:

I’ve successfully read in the penguins Excel file. Here’s a preview of the data:

species island bill_length_mm bill_depth_mm flipper_length_mm body_mass_g sex year
Adelie Torgersen 39.1 18.7 181 3750 male 2007
Adelie Torgersen 39.5 17.4 186 3800 female 2007
Adelie Torgersen 40.3 18.0 195 3250 female 2007
Adelie Torgersen NaN NaN - - - 2007
Adelie Torgersen 36.7 19.3 193 3450 female 2007

(and so on…) ) Note: Null values are represented as NaN (Not a Number) or ‘-’ for numerical columns, and as NULL for string columns.

bot("what tables do you have access to? give an overview of the data")

birdbrain:

Currently, I have access to one table named ‘penguins’.

Here’s an overview of the data in it:

  1. species: This is a string column that gives the species of the penguin.
  2. island: This is a string column that mentions the island the penguin is from.
  3. bill_length_mm: This is a float64 column that specifies the length of the penguin’s bill in millimeters. 4. bill_depth_mm: This is a float64 column that specifies the depth of the penguin’s bill in millimeters.
  4. flipper_length_mm: This is an int64 column that provides the length of the penguin’s flipper in millimeters.
  5. body_mass_g: This is an int64 column that gives the body mass of the penguin in grams.
  6. sex: This is a string column that mentions the sex of the penguin.
  7. year: This is an int64 column that gives the year of observation.

Please note that a string field can contain text, a float64 field can contain any real number (including decimal values), and an int64 field can contain any integer.

Interacting with the Internet via search

bot("what is this dataset? search the internet and find out more about its origin for me")

birdbrain:

The penguins dataset, a possible alternative to the classic Iris dataset, contains data on three species of penguins from three islands in the Palmer Archipelago, Antarctica [1][2][3].

The data comprises measurements such as bill length, bill depth, flipper length, body mass, along with categorical data such as species, island, and sex [3]. It provides a good mix of numeric and categorical data for various data exploration and machine learning tasks.

The dataset was originally collected and made available by Dr. Kristen Gorman and the Palmer Station, Antarctica, Long Term Ecological Research (LTER) [2].

The purpose of the Palmer LTER is to study the marine ecosystem in the western Antarctic Peninsula region as a microcosm of global ocean ecosystems, focusing on the processes that link the extent of annual sea ice to the biological dynamics of different trophic levels [2].

Sources: [1] Penguin dataset : The new Iris | Kaggle [2] (https://allisonhorst.github.io/palmerpenguins/) [3] Penguins Dataset Overview — iris alternative | by AbdulMajedRaja RS | Towards Data Science

Penguins Data Analysis Project

This repository contains an exploration and analysis of the penguins dataset. The data was originally collected and made available by Dr. Kristen Gorman and the Palmer Station, Antarctica, Long Term Ecological Research (LTER).

The penguins dataset provides detailed data on three species of penguins from three islands in the Palmer Archipelago, Antarctica. It includes measurements such as bill length, bill depth, flipper length, body mass, along with categorical data such as species, island, and sex. The dataset presents a rich mix of numeric and categorical data, making it an excellent resource for various data exploration and machine learning tasks.

Data

The data for this project is stored in an Excel file named ‘penguins.xlsx’ and is loaded into the project using Python. The data is then available for analysis and visualization.

Analysis

Detailed analysis on this data is yet to be performed. The objective is to derive meaningful insights about penguin species, their physical characteristics, and their distribution across the islands.

Contributing

Contributions are welcome. Please read the CONTRIBUTING.md for details.

License

This project is licensed under the terms of the MIT License.
